# Sharon Foulston — Speaker · Advocate · Author

Official website for Sharon Foulston, grief and identity speaker.

## About

Single-page speaker website built with HTML, CSS, and vanilla JavaScript. No frameworks or dependencies — fully self-contained.

**Brand direction:** Nautical without being literal.  
**Colours:** Navy `#0E2841` · Steel `#156082` · Orange `#E97132` · Cream `#F7F4EF`  
**Fonts:** Cormorant Garamond (headings) · Jost (body) · Space Mono (labels)

## Sections

- Hero with animated text reveal
- Empathetic entry cards ("What brought you here?")
- About Sharon
- The Return Framework™ — four stages
- Speaking topics & audiences
- Soundbite quote grid
- Book & Podcast resources
- Testimonials
- Booking CTA
- Footer

## Running locally

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Deployment

This is a static HTML file — deploy to any static host:

- **Netlify**: drag `index.html` to [netlify.com/drop](https://netlify.com/drop)
- **GitHub Pages**: enable Pages in repository Settings → Pages → Deploy from main branch
- **Vercel**: connect this repository in the Vercel dashboard
